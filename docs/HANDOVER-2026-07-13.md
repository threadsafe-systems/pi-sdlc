## Gotchas (verified this session)

- `node --test` silently skips missing test files (exit 0) — every PV1
  manifest for a task that ADDS a test file needs a `static.tests-exist`
  (`node --check <file>`) guard.
- A freshly `ensure-panel-agent`-stamped agent is NOT visible to the
  `subagent` tool mid-session; workaround: dispatch `delegate` with the
  generated body verbatim (pr-review/plan/spec agents ARE registered).
- The validator runner rejects `--report` paths outside the repo root.
- git `diff` pathspecs are cwd-relative — always use `:(top)<path>` for
  top-relative paths (see cleanAgainstHead in sdlc-status.mjs).
- Worktrees: create-then-enter via `tool:worktree_session` (implement:before
  hook); `npm ci` inside; dispose refuses ignored files — remove
  `node_modules` and stamped `.pi/agents/pi-sdlc-{pr-review,task-validate}.md`
  first.
- Merge needs owner-authorised `--admin` squash; PR title is the Conventional
  Commit that drives semantic-release.
